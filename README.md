# Hello there!

Welcome to my GitHub profile. I'm **Marco Saretta**, an industrial PhD researcher working on **energy system modelling**

## **Current Research**

My PhD project **"Commoditizing Green Hydrogen in Europe: From Efficient Market Design to Optimal Contracting and Investment"**
focuses on developing **quantitative tools** to:  
- **Price Power Purchase Agreements (PPAs)** and optimise contracts for green hydrogen.  
- Support **investment and contracting under uncertainty**.

## Main repositories
In my profile you'll find repositories on a range of topics within mathematical modelling. Here I listed six:

 1. **enlight** : EU power market model for system analysis
 2. **hydrogen-market-design** : Supplementary material for my publication on hydrogen market design
 3. **dtu-rl-hydro** : Personal repo exploring reinforcement learning for hydropower scheduling
 4. **methanol-plant-sizing** : Linear programming investment model for methanol plant sizing
 5. **pypsa-network-analyzer** : A package for faster analysis of PyPSA network files
 6.  **yahei-DTU/mlops_loadconsumption** : Load consumption forecasting with CNNs
